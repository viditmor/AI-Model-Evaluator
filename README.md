# AI Model Evaluation Platform

## Overview

The AI Model Evaluation Platform is a production-oriented web application developed to compare and evaluate responses generated by multiple Large Language Models (LLMs) available through Amazon Bedrock. The platform enables users to provide prompts, system instructions, expected (ground truth) responses, and output parsing rules, then evaluates each selected model using configurable quality metrics.

The project was designed as an internal tool to assist in selecting the most suitable foundation model for different AI use cases.

---

## Features

* Compare responses from multiple Amazon Bedrock foundation models.
* Evaluate model outputs against ground truth using configurable evaluation metrics.
* Support for user prompts, system prompts, and structured output parsing.
* Interactive dashboard built with React.
* FastAPI backend exposing REST APIs.
* Secure integration with Amazon Bedrock using AWS IAM credentials.
* Deployable on AWS EC2.

## Workflow

1. User enters:

   * Prompt
   * System Prompt (optional)
   * Ground Truth
   * Output Parser (optional)

2. User selects one or more Bedrock foundation models.

3. Backend invokes each selected model through Amazon Bedrock.

4. Generated responses are evaluated using configurable scoring metrics.

5. Individual metric scores and overall rankings are displayed in the dashboard.

## Installation

### Backend

```bash
cd backend

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

uvicorn main:app --reload
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
